Great Expectations

11:10 - 12:00

Great Expectations: What to expect when testing your data pipelines

Ensuring the reliability of modern data pipelines is challenging due to unpredictable data in production. This talk explores the historical struggles with testing data solutions, lessons learned, and advancements made through improved guidance and training.

We will discuss the adoption of Test-Driven Development (TDD) and Behavior Driven Development (BDD) to enhance code quality and align with business requirements. Despite these improvements, unpredicted behaviors still arise from variable data inputs in live environments. We'll explore strategies for mitigating these risks.

Additionally, we will delve into Data Observability and demonstrate the use of tools like pytest, behave, and Great Expectations for proactive monitoring and validation of data flows.

Join this session to gain insights into:

  • The historical challenges and lessons learned in testing data solutions
  • Applying TDD and BDD to improve code quality and business alignment
  • Techniques for managing unpredictable data behaviors in production
  • Leveraging Data Observability tools for continuous monitoring and improvement
